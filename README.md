# Fucking Awesome Image Editor

Application for photo editing

Support this methods:
1. Image rotation
2. Scaling
3. Color correction
4. Unsharped masking
5. Bilinear/bicubic filtration
6. Splains and lines
7. Retouch
8. Segmentation
9. A* visualization
